# 図表メモ

## 必要な図表・ダイアグラム

### 1. 歴史タイムライン
```
1958年 ─ John McCarthy (Lisp)
1964年 ─ Peter J. Landin (クロージャ発明)
1975年 ─ Sussman & Steele (Schemeでクロージャ定義)
1995年 ─ まつもとゆきひろ (Ruby誕生)
```

### 2. rb_block_t構造体の図解
```
rb_block_t
├── self: VALUE           (ブロック参照時のselfポインタ)
├── klass: VALUE          (現在のオブジェクトクラス)  
├── ep: VALUE*            (環境ポインタ) ←重要！
├── iseq: rb_iseq_t*     (YARV命令列)   ←重要！
└── proc: VALUE           (Procオブジェクト参照)
```

### 3. スタックフレーム変化の図
```
最初の状態:
┌─────────────────┐
│ トップレベル     │
│ スタックフレーム │
└─────────────────┘

10.times呼び出し後:
┌─────────────────┐
│ ブロック用       │
│ スタックフレーム │  ←新しく作成
├─────────────────┤
│ times メソッド   │
│ スタックフレーム │
├─────────────────┤  
│ トップレベル     │
│ スタックフレーム │
└─────────────────┘
```

### 4. メモリ配置図（スタック vs ヒープ）
```
スタック                    ヒープ
┌─────────────┐              ┌─────────────────┐
│ locals: str │ ──────────→ │ RString          │
│             │              │ "quick brown..." │
│ EP ─────────┼──┐          └─────────────────┘
└─────────────┘  │           
                 │           ┌─────────────────┐
                 └─────────→ │ rb_env_t        │
                             │ (ヒープコピー)   │
                             └─────────────────┘
```

### 5. Procオブジェクトの構造
```
VALUE ポインタ
    ↓
┌─────────────────┐
│ RTypedData      │
│ ├── RBasic      │
│ │   ├── flags   │
│ │   └── klass   │ ──→ Proc class
│ └── data ───────┼──→ rb_proc_t
└─────────────────┘       ├── rb_block_t
                          │   ├── iseq
                          │   └── EP
                          ├── envval
                          └── is_lambda
```

### 6. パフォーマンス比較グラフ

```
1.0 ┤
0.8 ┤      ██

```

### 7. クロージャの概念図
```
クロージャ = 関数 + 環境

┌─────────────┐   ┌─────────────┐
│   コード    │ + │    環境     │
│ (iseq)     │   │   (EP)     │
│ puts str   │   │ str = "..." │
└─────────────┘   └─────────────┘
        ↓
    rb_block_t
```
