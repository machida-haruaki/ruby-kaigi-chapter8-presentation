クロージャ概念図

クロージャ = ラムダ式 + 環境

┌─────────────┐   ┌─────────────┐
│  コード片   │ + │    環境     │
│  (iseq)     │   │   (EP)     │
│             │   │            │
│ puts str    │   │ str="..."  │
└─────────────┘   └─────────────┘
        ↓
   rb_block_t構造体

rb_block_t構造体:
├── iseq: rb_iseq_t*  (YARV命令列) ←「ラムダ式」
├── ep: VALUE*        (環境ポインタ) ←「環境」
├── self: VALUE       (selfポインタ)
├── klass: VALUE      (クラス情報)
└── proc: VALUE       (Procオブジェクト)

重要: EPがクロージャの核心機能を担っている
