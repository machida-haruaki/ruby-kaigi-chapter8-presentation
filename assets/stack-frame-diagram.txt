スタックフレーム変化図

最初の状態:
┌─────────────────┐
│ トップレベル     │
│ スタックフレーム │
│ locals: str     │
└─────────────────┘

10.times呼び出し後:
┌─────────────────┐
│ ブロック用       │ ←新しく作成
│ スタックフレーム │
│ locals: str2    │
├─────────────────┤
│ times メソッド   │
│ スタックフレーム │
├─────────────────┤  
│ トップレベル     │
│ スタックフレーム │ 
│ locals: str     │
└─────────────────┘

ポイント:
- メソッド呼び出しでスタックフレーム作成
- ブロック実行でもスタックフレーム作成
- EPは親スコープの変数にアクセス可能
